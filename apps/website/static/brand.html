Design contract

03 · Theme behavior

Dark is default

Visitors enter the site in soil-dark. Dark mode is the primary brand environment for homepage, blog, marketing pages, and DIEM economics.

defaultMode: dark · soil surface · green action

Light remains available

Light mode uses warm field-note surfaces with dark ink. It must stay readable, but it is not the default first impression.

optional toggle · warm cream · no pale mint text

04 · Rules

R1

Dark mode is the default. Soil-dark is the default site surface. Light mode is available and readable, but secondary.

R2

Green means network/action/ants. Use it for live network state, links, CTAs, pheromone paths, ant animation, and routing.

R3

Clay means money. Use clay/orange for $ANTS, $DIEM, USDC, fees, buy/burn, emissions, token pages, contracts, and economics sections.

R4

Use “fees,” not “revenues.” Economics copy should not imply equity, profit share, revenue share, or guaranteed return.

R5

Only one site theme switch. On Docusaurus pages, keep the built-in icon switch on the right. No extra “Dark / Light” text switch in the navbar.

R6

No invisible UI. Pale mint on cream is not allowed. Background decoration must never reduce readability.

R7

No giant ghost numbers. Section numbering can be small metadata, but not huge low-opacity background figures behind titles.

R8

Preserve content and SEO when redesigning. Visual updates should not rewrite page copy, titles, descriptions, canonical behavior, robots metadata, or structured content unless explicitly requested.
